# Rock_paper_scissors_cnb2.0 version-0.1
# Made with :heart: by Team EzPz
This is a project of Code'N'Build 2021. In which we have created a interesting way to play the old school "Rock, Paper and Scissors" game in a modern way with a twist!.

So you ask what we do in this ?
We have created an interractive web application that contains an animated character named 'Trump', which is a default player whih will be playing against the user and will react on the basis of match results.
